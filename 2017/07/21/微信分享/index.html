微信分享

**app分享内容设置

  1. app内容分享设置，app 里调用的不是js-sdk的方法，app里应该设置好了相关的功能，然后前端填值就可以了。然后app在根据页面里的内容读取相关信息

    let shareStr = `
<div class="wechat-share" style="display: none;width:0;height:0;overflow: hidden;">
<div style="display:none" id="__cxnewsapp_topbar">true</div>
 <div style="display:none" id="__cxnewsapp_orientation">flexible</div>
 <div style="display:none" id="__cxnewsapp_sharetext">${shareText}</div>
 <div style="display:none" id="__cxnewsapp_sharephotourl">${sharePhotoUrl}</div>
 <div style="display:none" id="__cxnewsapp_sharewxurl">${shareWxLink}</div>
 <div style="display:none" id="__cxnewsapp_sharewxthumburl">${shareWxImgUrl}</div>
 <div style="display:none" id="__cxnewsapp_sharewxtitle">${shareWxTitle}</div>
 <div style="display:none" id="__cxnewsapp_sharewxtext">${shareWxDesc}</div>
</div>
`

    上面字符串的变量应该不难理解

    • shareText 分享的描述, 非微信
    • sharePhotoUrl 分享的缩略图链接，需要以http://*** 的链接地址
    • shareWxLink 微信分享分享的链接地址
    • shareWxImgUrl 微信分享的缩略图
    • shareWxTitle 微信分享标题
    • shareWxDesc 微信分享描述

最后设置好上述的内容后需要将上述的内容添加到页面当中去

exp:

然而在单页应用里，每一个地址代表不同的页面，页面不同分享的内容也不同，意味着需要在每个路由下面分别去设置该页面对应的分享内容。
在Vue单页中，可以在vue-router的钩子 router.afterEach 之中去做相关简易的设置。一些需要根据网页内容动态生产分享内容页面可以做额外的设置。重点不在哪里设置，在于页面对应的路由下调用分享方法设置分享内容

微信web分享

微信web分享调用的是微信的js-SDK 所有预先得在页面中引入相关js并作相应的配置

  • 引入js

  • 调用配置方法

      getWXConfig () {
  let url = `http://api.caixin.com/wxsdk/wxconfig.php`
  let params = {
    url: window.location.href
  }
  this.$http.jsonp(url, {
    params: params,
    jsonp: 'callback'
  })
    .then((res) => {
      let data = res.body
      /* global wx:true */
      wx.config({
        debug: false,
        appId: data.appId,
        timestamp: data.timestamp,
        nonceStr: data.nonceStr,
        signature: data.signature,
        jsApiList: [
          'onMenuShareTimeline',
          'onMenuShareAppMessage',
          'onMenuShareQQ',
          'onMenuShareWeibo',
          'startRecord',
          'stopRecord',
          'onVoiceRecordEnd',
          'playVoice',
          'pauseVoice',
          'stopVoice',
          'translateVoice',
          'uploadVoice'
        ]
      })
    })
}

配置的方法只需要在页面加载中微信内容分享设置之前调用一次配置完即可，不需要再次配置

  • 方法同app，在每个需要分享的路由下调用分享方法设置分享内容

浏览器默认分享

  1. 浏览器默认自带分享不能设置相关描述
  2. 浏览器默认分享标题即页面中的title内容
  3. 浏览器默认分享到微信的缩略图默认是页面中第一张图片大于300*300的图片，图片的设置也需要有一定的技巧方法可参考https://www.caiyiming.com/article/201702231619.html
    原理也同上，在每个需要分享的路由下设置相应内容

在分享的页面中不要存在caixin同名的js变量名，可能是变量名冲突导致安卓分享实效
