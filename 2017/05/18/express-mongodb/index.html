express+mongodb

express 程序和 mongodb 的结构

mongo 配置文件 mongod.conf

systemLog:
    destination: file
    path: "/var/log/mongodb/mongod.log"
    logAppend: true
storage:
    dbPath: "/data/db"
net:
    bindIp: 127.0.0.1
    port: 27017

启动mongo

mongod –config /etc/mongod.conf
